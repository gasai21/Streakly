# 📱 Streakly

> Small habits, big streaks.

Streakly adalah aplikasi Android untuk membantu pengguna membangun kebiasaan harian, menjaga konsistensi, dan melacak progress melalui sistem streak yang sederhana namun powerful.

---

## ✨ Features

* ✅ Tambah dan kelola habit harian
* 🔥 Streak tracking (konsistensi harian)
* 📊 Statistik progress (harian & mingguan)

---

## 🧠 Tech Stack

* Kotlin
* Jetpack Compose (UI Toolkit)
* MVVM Architecture
* Room Database (Local Storage)

---

## 📱 Screens

* 🏠 Home (Today Habit)
* ➕ Add / Edit Habit
* ⚙️ On Boarding

---

## 🔄 App Flow

```
User opens app
   ↓
Set data user
   ↓
Home Screen (Today Habit)
   ↓
Checklist habit
   ↓
Progress & streak updated
   ↓
(Optional) View stats
```
